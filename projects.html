The system processes input images, performs preprocessing, and predicts disease categories with trained models. It helps farmers identify diseases at an early stage, improving crop quality and reducing losses. The project includes dataset preparation, model training, testing, and accuracy evaluation. This project strengthened knowledge in machine learning, computer vision, and data preprocessing techniques.

Technologies: Python, CNN, Machine Learning
